numpy.polynomial.hermite.Hermite.basis¶

static Hermite.basis(deg, domain=[-1, 1], window=[-1, 1])¶

Hermite polynomial of degree deg.

Returns an instance of the Hermite polynomial of degree d.

Parameters:

deg : int

Degree of the Hermite polynomial. Must be >= 0.

domain : array_like

The resulting array must be of the form [beg, end], where beg and end are the endpoints of the domain.

window : array_like

The resulting array must be if the form [beg, end], where beg and end are the endpoints of the window.

Returns :

p : Hermite instance

Notes

New in version 1.7.0.
