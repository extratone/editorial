numpy.broadcast.reset¶

broadcast.reset()¶

Reset the broadcasted result’s iterator(s).

Parameters:None :
Returns:None :

Examples

>>> x = np.array([1, 2, 3])
>>> y = np.array([[4], [5], [6]]
>>> b = np.broadcast(x, y)
>>> b.index
0
>>> b.next(), b.next(), b.next()
((1, 4), (2, 4), (3, 4))
>>> b.index
3
>>> b.reset()
>>> b.index
0
