27. Importing Modules¶

The modules described in this chapter provide new ways to import other Python modules and hooks for customizing the import process.

The full list of modules described in this chapter is:

  • 27.1. imp — Access the import internals
    • 27.1.1. Examples
  • 27.2. importlib – Convenience wrappers for __import__()
  • 27.3. imputil — Import utilities
    • 27.3.1. Examples
  • 27.4. zipimport — Import modules from Zip archives
    • 27.4.1. zipimporter Objects
    • 27.4.2. Examples
  • 27.5. pkgutil — Package extension utility
  • 27.6. modulefinder — Find modules used by a script
    • 27.6.1. Example usage of ModuleFinder
  • 27.7. runpy — Locating and executing Python modules
